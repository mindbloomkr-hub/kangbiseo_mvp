개인정보처리방침

본 방침은 강비서(이하 '회사')가 운영하는 '강비서'(이하 '서비스')를 이용하는 이용자의 개인정보를 보호하고, 관련 법령을 준수하기 위해 수립되었습니다.

1. 수집하는 개인정보 항목

서비스는 구글 OAuth 인증을 통해 다음과 같은 정보를 수집합니다.

  • 필수 항목: 이름(displayName), 이메일 주소, 프로필 이미지 정보
  • 선택 항목 (연동 시): Google 캘린더 일정 데이터 (제목, 시간, 장소)

2. 개인정보의 수집 및 이용 목적

수집된 정보는 다음의 목적을 위해서만 이용됩니다.

  • 이용자 식별 및 회원 관리
  • 강의 일정 통합 관리 기능 제공
  • 이동 시간 계산: 기존 일정 장소와 신규 일정 사이의 이동 시간을 연산하여 일정 수락 가능 여부 판단 서비스 제공

3. Google API 데이터 사용에 관한 준수 사항

'강비서'는 Google API로부터 받은 정보를 사용할 때 Google API 서비스 사용자 데이터 정책(제한적 사용 요구사항 포함)을 준수합니다.

  • 수집된 구글 사용자 데이터는 이용자에게 일정 관리 기능을 제공하는 용도 외에 제3자에게 판매하거나 광고 타겟팅에 활용하지 않습니다.
  • 이용자의 명시적인 동의 없이 시스템 외부로 데이터를 전송하지 않습니다.

4. 개인정보의 보유 및 파기

회사는 원칙적으로 이용자가 서비스에서 탈퇴하거나 개인정보 이용 동의를 철회하는 경우, 해당 정보를 지체 없이 파기합니다. 단, 관계 법령에 의해 보존할 필요가 있는 경우 해당 기간 동안 보관합니다.

5. 이용자의 권리

이용자는 언제든지 본인의 개인정보를 조회하거나 수정할 수 있으며, 서비스 탈퇴를 통해 개인정보 이용에 대한 동의를 철회할 수 있습니다.
